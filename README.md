# QUAL2000
Where I will be putting my QUAL2000 assignment work.

// How To Run: A C++ compiler will be required. Such as Visual Studio. This is an online compiler that should work: https://www.onlinegdb.com/online_c++_compiler
With the compiler open, or using the site, pressing the green play button or the run button should run the code.

// How To Run Tests: In the Main.cpp is where all the code to run the test will be. Everything is commented out to start. Each test group is separated. To run a specific test, delete the two '//' before the desired test and run. All of the tests are for parameters programmed in. 

If you wish to test the abort cases, you can open the TestMenu.cpp file and scroll to the functions with commented out lines of code. Remove the '//' in front and run to get the abort error for those cases.

// Trying Out The Application: If you wish to use the application as a user entering information, then remove the '//' before the:
- Menu mainMenu
- mainMenu.run();
lines and run the application again.
